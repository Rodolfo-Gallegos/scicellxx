* Dockerfile

** By using this file you can create a FULL container with spack and
   the packages listed in the "spack.yaml" file. No clean-up is
   performed.

* spack.yaml

** This file is used in the generation of the docker images. This file
   lists the packages that will be installed in the docker image.

** You can use this file to install packages in your local machine
   with help of Spack.
